株式会社翔泳社より刊行の書籍『WordPress 標準デザイン講座【第2版】』で利用する学習用サンプルファイルです。
本書中でサンプルファイルを使った学習があるLESSONでは、使用するファイルの場所をLESSON冒頭に掲載していますので、記載に従ってファイルを開き、学習を進めてください。

※なお、サンプルのHTMLファイルとCSSファイルは全て文字コードが[UTF-8 BOMなし]、改行コードが[LF]になっています。
編集の際は、必ず上記文字コードに対応したテキストエディタを使用してください。

※本書刊行後に判明した誤記載の訂正や補足情報などは、下記URLの翔泳社刊行物Q&Aページにて公開しています。ご参照ください。
http://www.shoeisha.co.jp/book/qa/
